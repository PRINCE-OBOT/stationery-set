# SVG Stationery Set

A SVG illustration project featuring a stationery-themed set including items like a pencil, ruler, light bulb, and other desk essentials.

Built to practice what I learnt from [The Odin Project SVG lesson](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-svg)

---

## Features

* Custom SVG illustrations
* Multiple stationery objects (pencil, ruler, bulb, etc.)
* Clean and scalable vector graphics
* Organized use of SVG elements and grouping
* Lightweight, resolution-independent design
* Easily reusable icons for UI or projects

---

## Tech Stack

* SVG

---

## What I Learned

* How SVG coordinate systems work
* Drawing shapes using paths, circles, and rectangles
* Using `viewBox` for scaling properly
* Grouping elements with `<g>` for organization
* Building reusable vector graphics without design tools
* Thinking in coordinates instead of pixels

---

## Challenges

* Positioning elements accurately using coordinates
* Keeping proportions consistent across objects
* Managing complex paths in a readable way
* Understanding how transforms affect grouped elements

---

## Notes

This project focuses purely on visual design using SVG syntax rather than interactivity or JavaScript logic.
